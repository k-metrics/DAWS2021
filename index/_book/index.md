# `rmdja` パッケージテンプレートの使い方

`bookdown` の機能をフルに使うには, 以下の点に注意してください.

1. "Knit" ボタンではなく Build ペーンから文書をビルドする
2. 最初に読み込むファイルは必ず `index.Rmd` という名前にする
